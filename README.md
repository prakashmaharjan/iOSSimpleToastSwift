# iOS Simple Toast Demo (Swift)

Apple does not have native toast UI control like android. There are lots or thirdparty dependency libraries to create Toast in iOS. But we can create one ourself easily.

If you don't want to use third party library and just want simple android like toast style in your app, then this is one of way to do so. 

I have used UILabel and UIView.animate with duration in order to mimic android like simple Toast.
